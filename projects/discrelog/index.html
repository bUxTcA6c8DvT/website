Discrelog

A discrete logic simulator.

The Product

This is a fun web app for a discrete logic simulation game. You can drag elements to the screen and make connections between components to create working circuites. This is a game created by a hobbyist developer geared towards those with interest in circuitry. This project is aligned towards the niche of super users of technology and gamers alike. The ultimate goal of this project was to update the UI to be clean and fresh, and is an example of visual design work.

The Solution

  • Provide light and dark themes that complement each other
    Light and dark themes are popular among the developer community. Providing both adds a feel of customization to the app.
  • Graph paper feel to match the game schematic
    The visual aspect was made to match physical graph paper. Getting back to the games roots as circuit drawings made on graphing paper.
  • Unique visual style for the game
    The game has a unique visual style to draw attention to it and to help it stand out amongst other web apps.
  • Visual dragging bars at the top of windows to make resizing stand out.
    One aspect of the UI that is important to the developer is draggable sub-windows. Clear and clean bars at the top of each window help discern and signify each as draggable.
  • Large tutorial section on the home page
    Tutorials are a great way to learn how to play the game. Tutorials are a large part of the onboarding process so a large section for dedicated tutorials helps draw your attention.

Conclusion

This was a very fun project. Tying the design into a physical medium like graph paper really helps to give it personality. It may be necessary to provide a more thorough onboarding flow to teach users how to start building with the game.
